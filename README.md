Uma aplicação web desenvolvida para ser um guia de referência sobre gêneros de jogos. O projeto visa fornecer a criadores de jogos e entusiastas informações claras e objetivas sobre o que define cada gênero, suas mecânicas principais e características essenciais. Uma ferramenta prática para auxiliar no estudo, planejamento e desenvolvimento de novos jogos, garantindo uma base sólida sobre os conceitos de cada estilo.
